# Movie Finder

Movie finder created with React router.
Searh for your favorite movie.

## Technologies used
* React
* react-redux 
* react-router
* bootstrap
* webpack
* axios
* chai
* nightmare
* Express

## Setup
Clone/download the repo onto your local system then run the following commands:
```
npm install
```
```
npm run build
```
```
npm start
```
Then go to http://localhost:3000 in your browser

## To test
```
npm run build
```
```
npm test
```
